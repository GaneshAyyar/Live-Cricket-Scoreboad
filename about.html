About This Project

This Cricket Scoreboard Web App allows users to:

  • ➕ Add players to the batting order
  • 📝 Update individual player's score ball-by-ball
  • 🏏 Score boundaries easily (4s, 6s)
  • ☠️ Mark players out
  • 📋 View live scoreboard and commentary

Built using HTML, CSS, and JavaScript.
Designed for cricket fans who want to simulate or record real-time matches!
